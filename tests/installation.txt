Installation 

- install Python 2.x 

- install selenium for python: 

  Windows: 
  newer version of python alreaedy contain pip. 
  It is located in a subdirectory (scripts).
 
  cmd -> cd c:\....\Python..\scripts
         pip install selenium

- install geckodriver (https://github.com/mozilla/geckodriver/releases) and add path to PATH variable
  (you can also copy the geckodriver into the python directory without modifying PATH variable)
